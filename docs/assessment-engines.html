1. Security Compliance Assessment

Agent: run_assessment CLI: run_assessment.py

Full multi-framework compliance assessment across all 10 evaluation domains with 68 collectors, 113 check functions, and up to 11 frameworks (525 controls).

What it does:

  • Discovers all Azure subscriptions and Entra ID configuration
  • Runs 68 collectors concurrently to gather 218 evidence types
  • Evaluates against 113 check functions across 10 domains
  • Maps findings to selected compliance frameworks
  • Generates reports in up to 8 formats
python run_assessment.py --tenant <id> --framework NIST-800-53 CIS FedRAMP

2. Risk Assessment

Agent: analyze_risk CLI: run_risk_analysis.py

Security risk gap analysis across 4 categories: Identity, Network, Defender, and Configuration.

What it does:

  • Identity risks: MFA gaps, stale accounts, risky users, privilege escalation paths
  • Network exposure: open NSGs, public storage, missing firewalls
  • Defender coverage: missing plans, disabled auto-provisioning
  • Config hygiene: unpatched resources, missing encryption, policy drift
python run_risk_analysis.py --tenant <id>

3. Data Security Assessment

Agent: assess_data_security CLI: run_data_security.py

Data-layer posture assessment covering 12 categories.

What it does:

  • Storage exposure (public containers, blob access, SAS tokens)
  • Database security (SQL audit, TDE, firewalls, Cosmos DB)
  • Encryption posture (in-transit, at-rest, Key Vault expiry)
  • Data classification (Purview labels, sensitivity coverage)
  • Messaging security (Service Bus, Event Hub)
  • Cache security (Redis access, TLS)
python run_data_security.py --tenant <id>

4. RBAC Assessment

Agent: generate_rbac_report CLI: run_rbac_report.py

Interactive RBAC hierarchy visualization with PIM and risk analysis.

What it does:

  • Builds full hierarchy: Management Groups → Subscriptions → Resource Groups
  • Shows all role assignments with scope, principal, and role definition
  • Expands group memberships to show effective access
  • Identifies PIM-eligible vs. active assignments
  • Flags high-risk assignments
python run_rbac_report.py --tenant <id>

5. Copilot Readiness Assessment

Agent: assess_copilot_readiness CLI: run_copilot_readiness.py

M365 Copilot readiness assessment with 9+ categories.

What it does:

  • Oversharing analysis (broad SharePoint/OneDrive access)
  • Sensitivity label coverage
  • DLP policy coverage and gaps
  • Restricted SharePoint search configuration
  • Access governance (reviews, entitlement, guest policies)
  • Conditional access readiness
  • Information barriers
  • Audit and compliance
python run_copilot_readiness.py --tenant <id>

6. AI Agent Security Assessment

Agent: assess_ai_agent_security CLI: run_ai_agent_security.py

Security posture for 6 AI platforms across 23+ assessment areas.

What it does:

  • Copilot Studio (bot auth, DLP connectors, tenant isolation)
  • Azure AI Foundry (model deployment, endpoint auth, network isolation)
  • Custom agents (API auth, rate limiting, input validation)
  • AI model governance (registry, approvals, version control)
  • Data handling (training data, inference logging, PII)
  • Responsible AI (content filtering, grounding, prompt injection)
python run_ai_agent_security.py --tenant <id>

7. PostureIQ Assessment

Agent: run_postureiq_assessment AI-Powered

Risk-weighted posture assessment with attack path analysis and AI fix recommendations.

What it does:

  • Applies risk-weighted scoring (severity, exploitability, blast radius)
  • Identifies attack paths (chains of findings that create exploitable paths)
  • Ranks findings by fix priority for maximum risk reduction
  • Generates AI-powered remediation with step-by-step instructions
